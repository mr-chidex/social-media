## API for a Social Media

A Node JS RESTful API. Socal Media ApI

## Built With

- Node.js
- Mongo DB
- Express
- Git
- Json Web Token
- Postman
- TypeScript

## Clone this project

```
git clone https://github.com/mr-chidex/social-media.git
```

```
cd social-media
```

## Install dependencies

```
yarn install
```

### Running this project locally

```
yarn dev
```
